# sql-core-concepts-practice
SQL PRACTICE REPOSITORY

This repository contains my practice and implementation of core SQL concepts using structured queries on sample datasets. It covers fundamental database operations across multiple categories:

### Topics Covered

### 1. Data Query Language (DQL)

* SELECT, FROM
* WHERE (filtering records)
* GROUP BY (data grouping)
* HAVING (filtering grouped data)
* ORDER BY (sorting results)
* LIMIT (restricting output)

---

### 2. Aggregate Functions

* COUNT()
* SUM()
* AVG()
* MIN()
* MAX()

Used to perform calculations and summarize data.

---

### 3. Data Manipulation Language (DML)

* INSERT – Add new records
* UPDATE – Modify existing records
* DELETE – Remove records

---

### 4. Data Definition Language (DDL)

* CREATE – Create tables
* ALTER – Modify table structure

  * ADD
  * DROP
  * RENAME
  * CHANGE
  * MODIFY
  * TRUNCATE – Remove all records from a table

---

## Purpose

The goal of this repository is to:

* Build a strong foundation in SQL
* Practice real-world database operations
* Understand how to manage and query relational data efficiently
